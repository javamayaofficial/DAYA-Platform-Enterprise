# UI

## Layar
- Register
- Login
- Forgot Password
- Reset Password
- Verify Notice
- Verify Result
- Security Dashboard
- Device Sessions
- Login History
- Admin Users
- Admin User Roles

## State Wajib
- Success flash
- Validation error
- Empty state untuk tabel bila data belum ada
- Protected state untuk area auth/admin
