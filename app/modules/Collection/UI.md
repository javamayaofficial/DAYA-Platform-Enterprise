# UI

## Prinsip
- Tetap memakai layout Bootstrap 5 admin yang sudah ada.
- Tidak mendesain ulang shell framework.
- Menjaga struktur mobile-first, card-driven, dan table-driven.

## Layar
- Dashboard Collection: ringkasan collection creator dan shortcut aksi.
- Create Collection: form metadata container.
- Edit Collection: form revisi metadata container.
- Collection Detail Owner: detail container, publish/draft, tambah item, hapus item, dan reorder item.
- Collection Directory Admin: filter, list, dan monitoring collection.
- Collection Detail Admin: audit metadata collection dan item aktif.
- Public Collection Directory: listing collection published.
- Public Collection Page: halaman publik collection berbasis slug.
