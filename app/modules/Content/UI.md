# UI

## Prinsip
- Tetap memakai layout Bootstrap 5 admin yang sudah ada.
- Tidak mendesain ulang shell framework.
- Menjaga struktur mobile-first, card-driven, dan table-driven.

## Layar
- Dashboard Content: ringkasan karya creator dan shortcut aksi.
- Create Content: form metadata, access policy, SEO, dan body karya.
- Edit Content: form revisi karya.
- Content Detail Owner: detail karya, statistik, dan pengelolaan `Content Part`.
- Content Directory Admin: filter, list, dan moderasi karya.
- Content Detail Admin: review dan audit karya.
- Public Content Directory: pencarian karya published.
- Public Content Page: halaman publik SEO-friendly berbasis slug.
