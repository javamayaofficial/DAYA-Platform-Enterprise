# UI

## Prinsip
- Menggunakan layout Bootstrap 5 global yang sudah ada.
- Tidak mendesain ulang shell layout framework.
- Komponen difokuskan pada card, table, form, dan status summary yang konsisten dengan Authentication.

## Layar
- Dashboard Creator: ringkasan identity utama, verification, dan shortcut aksi.
- Registrasi Creator: form gabungan identity, profil, multi category, multi skill, KYC, dan setting publik.
- Profil Creator: detail identity, statistik, social links, typed portfolio, achievement, badge, dan danger zone.
- Edit Creator: form identity dan profile enrichment.
- Admin Directory: filter, list, detail, dan review verification/statistics/badge.
- Public Directory: list Creator aktif dengan search dan kategori.
- Public Show: profil publik SEO friendly, badge, skill, achievement, social links, dan portfolio publik.
